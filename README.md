# PRODIGY_DS_03
Decision Tree Classifier for predicting customer purchase behavior using the Bank Marketing Dataset.

### Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data.

---

## Dataset

- Dataset: Bank Marketing Dataset
- Source: UCI Machine Learning Repository
- File Used: bank.csv

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## Steps Performed

### 1. Data Loading
- Loaded the Bank Marketing dataset using Pandas.

### 2. Data Exploration
- Displayed the first five rows.
- Checked dataset information.
- Verified missing values.

### 3. Data Preprocessing
- Converted categorical columns into numerical values using Label Encoding.
- Split the dataset into training and testing sets.

### 4. Model Building
- Built a Decision Tree Classifier using Scikit-learn.
- Trained the model on the training dataset.

### 5. Model Evaluation
- Predicted customer purchase decisions.
- Calculated model accuracy.
- Generated a Confusion Matrix.

### 6. Visualization
- Decision Tree Diagram
- Confusion Matrix
- Feature Importance Graph

A Decision Tree Classifier was successfully built to predict customer purchase behavior. The model was trained and evaluated using the Bank Marketing dataset. Various visualizations were created to understand the model's performance and the importance of different features.
